# CalFrac: Calcaneal Fracture Dataset and Code

This repository provides the source code and data access protocol for the calcaneal fracture segmentation project.
This repository provides the official implementation of the paper:Edge-Enhanced Calcaneal Fracture Segmentation Using a
Multi-task CNN-Transformer Hybrid Network

## Overview

Calcaneal fractures are among the most common and complex injuries of the foot, often presenting with irregular fracture patterns and subtle structural variations. Accurate segmentation of fracture regions from CT scans is crucial for clinical diagnosis, surgical planning, and post-operative evaluation.
This project provides:
- Deep learning implementation for calcaneal fracture segmentation
- The CalFrac dataset access protocol
- Training and inference scripts for reproducibility
